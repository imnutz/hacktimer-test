Blink expensive background timer throttling demo.

This page contains a task which busy-waits for 50ms and runs in cycle every 500ms.

  • When page is not throttled (e.g. page is foregrounded), interval between tasks should be around 500ms.
  • When page is backgrounded and expensive background timer throttling is not enabled, interval between tasks should be around 1000ms.
  • When expensive background timer throttling is enabled, interval between tasks should be around 5000ms.
